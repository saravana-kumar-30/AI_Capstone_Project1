HOW IT WORKS
================================================================================

PROCESS FLOW:
  1. User submits loan application via Streamlit UI
  2. FastAPI receives and validates the data
  3. Orchestrator coordinates 4-stage agent workflow:
     
     Stage 1: Applicant Profile Agent (analyzes income, employment, history)
              ↓
     Stage 2: Financial Risk Agent (calculates DTI, credit risk, anomalies)
              ↓
     Stage 3: Loan Decision Agent (synthesizes decision, calculates risk score)
              ↓
     Stage 4: Compliance Agent (logs decision, generates case ID, audits)
  
  4. Final decision returned to UI with explanation
  5. All data stored in audit trail

DECISION OUTPUT:
  • Classification: Approve / Reject / Review
  • Risk Score: 0-100
  • Confidence Level: 0-100%
  • Key Factors: List of decision drivers
  • Explanation: Detailed reasoning
  • Case ID: Unique identifier for tracking

================================================================================
RISK SCORING ALGORITHM
================================================================================

The system calculates risk based on multiple factors:

Credit Score Analysis:
  • Score < 650  → +30 points (very risky)
  • Score < 700  → +15 points (below average)

Debt-to-Income Ratio:
  • DTI > 0.5    → +25 points (too much debt)
  • DTI > 0.4    → +10 points (moderate debt)

Employment Risk:
  • High risk    → +20 points
  • Medium risk  → +5 points

Loan Amount Risk:
  • Loan-to-income > 5x → +20 points

Income Stability:
  • Score < 50   → +15 points

DECISION RULES:
  • Score 0-45    → 🟢 APPROVE (low risk)
  • Score 45-70   → 🟡 REVIEW (manual review needed)
  • Score 70-100  → 🔴 REJECT (high risk)
